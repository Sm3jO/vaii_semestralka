Assassin's Creed Mirage

Rating: 7/10

Published on: October 11, 2023

  • Positives:
  • The atmosphere of Baghdad
  • Nostalgia for old mechanics
  • Voluntary RPG system
  • Negatives:
  • Weak storyline
  • Technical problems
  • Very unengaging tasks

A great return to the roots, honoring tradition, and reviving beloved game mechanics. Finally, there is hope for Assassin's Creed series veterans that after the extremely stretched open-world RPG installments, the developers will return to the series' foundations that we all remember so fondly. In this regard, it must be said that Mirage is doing a good job.
